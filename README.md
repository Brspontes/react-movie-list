<h2>Usage</h2>
First clonning repository using

```bash
git clone https://github.com/Brspontes/react-movie-list.git

cd react-movie-list
```
<br />
In current folder, before compilation, restore packages with

```bash
npm install
or
yarn install
```
Run App
```bash
npm run dev
or
yarn dev
```

Run Mocked Server
```bash
npm run server
or
yarn server
```

Go to:  https://localhost:8080/
